# AstrBot 运动组手插件

这是一个用于 AstrBot 的运动助手插件，支持自动同步每日运动步数到主流健康运动平台。

## 功能特性

- 🏃‍♂️ 支持同步步数到主流健康运动平台
- 🔧 可配置默认步数
- 🛡️ 参数验证和错误处理
- 📱 友好的用户交互界面
- ⚡ 异步处理，响应迅速

## 安装方法

1. 将插件文件夹放置到 AstrBot 的插件目录中
2. 重启 AstrBot 或重新加载插件
3. 插件将自动加载并可以使用

## 使用方法

### 基础命令

- `/步数 邮箱 密码 步数` - 修改指定账户的步数
- `/步数 邮箱 密码` - 使用默认步数修改
- `/步数帮助` - 查看详细使用说明

### 支持的别名

- `/修改步数`
- `/运动步数`

### 使用示例

```
/步数 example@qq.com mypassword 15000
/步数 user@gmail.com password123
/步数帮助
```

## 配置说明

插件支持以下配置项：

- `api_url`: API接口地址（默认：https://api.nycnm.cn/api/v2/zepplife ）
- `api_key`: API密钥（如果需要的话）
- `timeout`: 请求超时时间（默认：10秒）
- `default_steps`: 默认步数（默认：20000）

## 注意事项

- 仅支持私聊使用，群组中不会响应任何命令
- 步数范围：0-98800（未设置则随机 20000-30000）
- 请确保账号邮箱和密码正确
- 支持主流健康运动平台账户
- 修改后可能需要等待几分钟同步
- 插件不会保存您的账户信息
- 所有数据仅用于API调用
